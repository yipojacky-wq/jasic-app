# JASIC App 示範操作手冊

最後更新：2026-07-13

本手冊用於展示 JASIC 股票分析工具 MVP。JASIC 是研究與風險檢核工具，不保證獲利、不提供自動下單、不作為個人化投資顧問。

## 1. 示範網址

公開 PWA：

```text
https://yipojacky-wq.github.io/jasic-app/
```

建議使用：

- 手機 Safari / Chrome
- 桌機 Chrome / Edge

若手機看到舊畫面，請重新整理，或關閉分頁後重新開啟網址。

## 2. 示範前確認

開始示範前，先確認：

1. 已登入 JASIC。
2. 底部有五個頁籤：
   - 1 總經
   - 2 選股
   - 3 AI
   - 4 追蹤
   - 5 報告
3. 右下或底部可看到免責文字：
   - JASIC 為研究與風險檢核工具，不保證獲利，不提供自動交易。

## 3. 建議示範順序

建議照以下順序展示，整體約 5 至 8 分鐘。

```text
總經 Dashboard → 三層漏斗選股 → AI 投資檢核 → 個人追蹤 → 趨勢報告
```

## 4. 頁面一：總經 Dashboard

點選底部：

```text
1 總經
```

示範重點：

- Market Score / 市場分數
- 市場燈號
- 五大總經指標
- AI 市場摘要
- 今日風險雷達

可使用說明話術：

```text
這一頁先看市場環境，JASIC 不會直接叫使用者買賣，而是先判斷目前市場是否適合承擔風險。
```

若看到「中性輪動」：

```text
代表目前市場不是全面風險偏多，也不是全面風險偏空，較適合用選股與風險控管來分層操作。
```

## 5. 頁面二：三層漏斗選股

點選底部：

```text
2 選股
```

示範重點：

- 三層漏斗：
  1. 市場環境
  2. 法人 / 主力 / OI
  3. 技術 / 風險
- 今日 Top 20 候選股
- 燈號、風險、類型、排序篩選
- 匯出 CSV

建議操作：

1. 先不要輸入搜尋，直接往下滑看 Top 20。
2. 點一檔股票卡片，展開三層漏斗證據。
3. 點「開啟個股作戰室」查看個股分析。

如果搜尋某股票沒有結果：

```text
代表該股票不在今日 Top 20 候選股，不是系統壞掉。可按「顯示全部」回到完整清單，或到 AI 檢核直接輸入該股票。
```

可使用說明話術：

```text
這裡不是全市場報價表，而是 JASIC 根據目前資料挑出的研究候選清單。Top 20 是候選，不是買進建議。
```

## 6. 頁面三：AI 投資檢核

點選底部：

```text
3 AI
```

示範輸入範例：

```text
股票代號：2330
平均成本：980
張數：1
投資期間：短線 / 波段 / 中期 / 長期
風險偏好：穩健
```

點選：

```text
開始 AI Check
```

輸出會包含：

- 結論
- 原因
- 風險
- 建議
- 信心度

可使用說明話術：

```text
AI Check 不會保證獲利，也不會自動交易。它會把市場、個股分數、風險與使用者持股條件整理成一份研究檢核建議。
```

如果結果是「加碼」：

```text
這不是叫使用者直接買進，而是代表規則檢核允許保守評估加碼，仍需受風險控管約束。
```

如果結果是「觀望」：

```text
代表目前資料、信心或市場條件還不夠支持更積極的動作。
```

## 7. 頁面四：個人追蹤

點選底部：

```text
4 追蹤
```

示範重點：

- Watchlist / 我的觀察清單
- 研究持倉
- 分數變化
- 風險警示
- 個人化警示規則

可使用說明話術：

```text
這裡是個人化追蹤區，用來集中看自己關心的股票、部位風險與分數變化。
```

## 8. 頁面五：趨勢報告

點選底部：

```text
5 報告
```

目前會看到 4 種報告：

- 每日市場
- 核心池週報
- 個股戰情
- 風險警示

建議操作：

1. 點「全部」查看所有報告。
2. 點「每日市場」查看市場摘要。
3. 點「核心池週報」查看候選池概況。
4. 點「個股戰情」查看單一股票報告。
5. 點「風險警示」查看風險重點。

可使用說明話術：

```text
報告頁把 Dashboard、候選池與個股分析整理成固定格式，方便每日或每週追蹤，而不是只看單次 AI 回答。
```

## 9. 手機操作注意事項

若畫面滑動不順：

1. 先重新整理頁面。
2. 若仍異常，關閉 Safari 分頁後重新開啟。
3. 若安裝成主畫面 PWA，建議從主畫面圖示開啟。

若資料沒有更新：

1. 下拉或點重新整理。
2. 等待 5 至 10 秒。
3. 重新登入一次。

若報告或候選股看起來不足：

```text
請先按重新整理。JASIC 後端會依正式行情與已產生的分數快照回傳資料。
```

如果只有 3 分鐘，可照這個順序：

1. 打開總經頁：
   - 看市場分數與中性輪動。

   - 看 Top 20 候選股。
3. 打開 AI 頁：
   - 輸入 2330 / 成本 980 / 1 張 / 穩健。
4. 打開報告頁：
   - 展示每日市場、核心池週報、個股戰情、風險警示。
5. 結尾說明：
   - 不保證獲利、不自動交易、僅供研究與風險檢核。
